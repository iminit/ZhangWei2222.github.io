  1. 什么是集合？有什么用？

    • 数组其实就是一个集合，集合实际上就是一个容器，可以来容纳其他类型的数据。

    • 集合为什么说在开发中使用较多？

      • 集合是一个容器，是一个载体，可以一次容纳多个对象。
      • 在实际开发中，假设连接数据库，数据库当中有10条记录，那么假设把这10条记录查询出来，在 Java 程序中会将10条数据封装成10个 Java 对象，然后将10个 Java 对象放到某一个集合当中，将集合传到前端，然后遍历集合，将一个数据一个数据展现出来。
