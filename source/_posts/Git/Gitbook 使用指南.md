使用 Gitbook

```shell
$ git clone ...
$ gitbook install
$ gitbokk serve # 本地预览
$ gitbook build # 先 build 再 push，就发布了
$ git add . # git status时会有很多历史改动，不要紧张，不太需要管，确实生成
$ git commit -m ''
$ git pull
$ git push

# 对于| & # 可以使用code值进行转义，比如&#124; -》 |
```
